# SPEC.md - 仕様書

## アプリ概要

講義や自習の音声を入力すると、文字起こし・要約・要点を自動生成するスマホ対応Webアプリ。

## ユーザー導線

1. Gemini APIキーを入力して保存
2. 講義名を入力（任意）
3. 音声ファイルを選択 または その場で録音
4. 「文字起こし・要約を生成」ボタンを押す
5. 文字起こし・要約・要点が表示される
6. 各結果をコピーできる

## 入力

| 項目 | 必須 | 説明 |
|------|------|------|
| Gemini APIキー | 必須 | localStorageに保存。GitHubには含まない |
| 講義名 | 任意 | テキスト入力 |
| 音声 | 必須 | ファイル選択 または ブラウザ録音 |

## 出力

| 項目 | 説明 |
|------|------|
| 文字起こし | 音声の全文テキスト |
| 要約 | 3行程度の要約 |
| 要点 | 箇条書き3〜5個 |

## エラー処理

- APIキー未入力 → 「APIキーを入力してください」
- 音声未選択 → 「音声ファイルを選択してください」
- API失敗 → エラーメッセージを表示
- マイクアクセス拒否 → 「マイクのアクセスを許可してください」

## 技術構成

- フロントエンドのみ（HTML/CSS/JS）
- GitHub Pagesで公開
- Gemini API（gemini-2.0-flash）で文字起こし・要約
- APIキーはlocalStorageに保存

## 対応環境

- PC・スマホ両対応（レスポンシブ）
- Chrome / Safari / Edge
